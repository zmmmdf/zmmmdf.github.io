Number of hanging vertices 1

Statement:

( The original statement can be found here )Given a simple undirected unweighted graph. Count the number of hanging vertices in it. The vertex is hanging, if its degree is 1 .

Input:

The first line contains number n~(1 \le n \le 1000). The next n lines contain the adjacency matrix.

Output:

Given a simple undirected unweighted graph. Count the number of hanging vertices in it. The vertex is hanging, if its degree is 1.

Constraints:

Time limit 3 seconds
Memory limit 128 MiB

Solution:

def count_leaves(n, graph):
    leaves = 0
    for i in range(n):
        degree = sum(graph[i])
        if degree == 1:
            leaves += 1
    return leaves

n = int(input())
graph = []
for _ in range(n):
    row = list(map(int, input().split()))
    graph.append(row)

result = count_leaves(n, graph)
print(result)
