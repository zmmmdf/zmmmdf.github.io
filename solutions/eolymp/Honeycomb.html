Honeycomb

Statement:

( The original statement can be found here )The bee can go in honeycomb as shown in the figure — with moves 1 and 2 from upper row and with move 3 from the lower.

Input:

The number of hexagons n~(1 \le n \le 45) in the upper row. The lower row contains 1 hexagon less.

Output:

The bee can go in honeycomb as shown in the figure — with moves 1 and 2 from upper row and with move 3 from the lower.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Show problem classificationcombinatorics, dynamic-programming

Solution:

#include <bits/stdc++.h>
using namespace std;
int main() {

    long long n,i;

    cin>>n;

    long long a[n+5];
    a[0]=1;
    a[1]=1;
    a[2]=2;

    for(i=3;i<n;i++){
        a[i]=a[i-1]+a[i-2];
    }

    cout<<a[n-1];
    return 0;
}
