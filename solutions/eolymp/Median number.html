Median number

Statement:

( The original statement can be found here )Three different integers a, b, c are given. Print the median number.

Input:

Three integers a, b, c that do not exceed 1000 by absolute value.

Output:

Three different integers a, b, c are given. Print the median number.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Show problem classificationif-then

Solution:

a, b, c = input().split()

if(abs(int(a)) < 1000 and abs(int(b)) < 1000 and abs(int(c)) < 1000):
    if(int(a) < int(b) < int(c) or int(c) < int(b) < int(a)):
        print(b)
    elif(int(a) < int(c) < int(b) or int(b) < int(c) < int(a)):
        print(c)
    elif(int(b) < int(a) < int(c) or int(c) < int(a) < int(b)):
        print(a)
