Statement:

( The original statement can be found here )Fibonacci numbers is a sequence of numbers F( n ), given by the formula: F( 0 ) = 1 , F( 1 ) = 1 , F( n ) = F( n - 1 ) + F( n - 2 ) Fibonacci number f is given. Find its index n . You must find such index n that F( n ) = f .

Input:

Fibonacci number f (2 ≤ f ≤ 2·10^9).

Output:

Fibonacci numbers is a sequence of numbers F(n), given by the formula:

Constraints:

Time limit 1 second
Memory limit 128 MiB

Solution:

n = int(input())
a1, a2 = 0, 1
for i in range(n):
    f = a1 + a2
    a1 = a2
    a2 = f
    if n == f:
        print(i + 1)
        break
