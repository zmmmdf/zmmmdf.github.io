Vowels doubling

Statement:

( The original statement can be found here )A string consists of lowercase Latin letters and spaces. Double all vowels in it, that is, the letters a, e, i, o, u and y .

Input:

One line, consisting of lowercase Latin letters and spaces. The line contains no more than 1000 symbols.

Output:

A string consists of lowercase Latin letters and spaces. Double all vowels in it, that is, the letters a, e, i, o, u and y.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Author Matviychuk Sergiy Volodymyrovych
Source "ABC programming"

Solution:

#include<bits/stdc++.h>
using namespace std; int main(){ string s; getline (cin,s); for(int i=0;i<s.size();i++){ cout<<s[i]; if(s[i]=='a'){ cout<<"a"; } if(s[i]=='e'){ cout<<"e"; } if(s[i]=='u'){ cout<<"u"; } if(s[i]=='i'){ cout<<"i"; } if(s[i]=='o'){ cout<<"o"; } if(s[i]=='y'){ cout<<"y"; }

}
}
