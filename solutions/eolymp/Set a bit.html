Set a bit

Statement:

( The original statement can be found here )Two integers a and k are given. Print the number that is obtained from а by setting the value of k -th bit to 1 .

Input:

One line contains two integers a and k (0 ≤ a ≤ 10^9). The least significant bit of a is a bit number zero.

Output:

Two integers a and k are given. Print the number that is obtained from а by setting the value of k-th bit to 1.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source 2010 LKSH Berendevy Polyany, August, Parallel C, Day 1, Problem B
Show problem classificationbitwise-operators

Solution:

#include <bits/stdc++.h>
using namespace std;

int main() {
  ios_base::sync_with_stdio(0);
  cin.tie(0);

  int n, k;
  cin >> n  >> k;
  cout << (n | (1<<k)) << endl;

  return 0;
}
