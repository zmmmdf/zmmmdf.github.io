Fibonacci strings generation

Statement:

( The original statement can be found here )Generate n -th Fibonacci string that is defined with the next recurrent formula: For example, f(3) = f(2) + f(1) = (f(1) + f(0)) + f(1) = "b" + "a" + "b" = "bab" .

Input:

One integer n~(0 \le n \le 20).

Output:

Generate n-th Fibonacci string that is defined with the next recurrent formula:

Constraints:

Time limit 1 second
Memory limit 128 MiB
Author Mykhailo Medvediev

Solution:

#include <bits/stdc++.h>
using namespace std;

string f(int i){
    if (i==0) return "a";
    else if(i==1) return "b";
    else return f(i-1) + f(i-2);
}

int main() {

    int n;

    cin >> n;

    cout << f(n);
    return 0;
}
