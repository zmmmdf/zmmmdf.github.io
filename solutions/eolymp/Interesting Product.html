Interesting Product

Statement:

( The original statement can be found here )Find all possible values of product i \cdot j , if the integer values of variables i and j vary for i from a to b and j from c to d .

Input:

One line contains 4 numbers a, b, c and d (1 \le a, b, c, d \le 10, a can be greater than b, c can be greater than d).

Output:

Find all possible values of product i \cdot j, if the integer values of variables i and j vary for i from a to b and j from c to d.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source SFE-2010 Variant 16
Show problem classificationbeginners

Solution:

a, b, c, d = map(int, input().split())

if a > b:
    a, b = b, a
if c > d:
    c, d = d, c

st = set()
for i in range(a, b + 1):
    for j in range(c, d + 1):
        st.add(i * j)

print(len(st))
