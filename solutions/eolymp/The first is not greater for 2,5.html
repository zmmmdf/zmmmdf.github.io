The first is not greater for 2,5

Statement:

( The original statement can be found here )Array of real numbers is given. Find the first element in array which value does not exceed 2.5 .

Input:

In the first line the number of elements in array n~(0 < n \le 100) is given. Next line describes n real numbers.

Output:

Array of real numbers is given. Find the first element in array which value does not exceed 2.5.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source SFE-2010 Variant 7
Show problem classificationbeginners

Solution:

def solve() -> None:
    n = int(input())
    success = False
    cur = list(map(float, input().split()))

    for i in range(n):
        if cur[i] <= 2.5:
            print(f"{i+1} {cur[i]:.2f}")
            success = True
            break

    if not success:
        print("Not Found")

solve()
