Sum of GCD

Statement:

( The original statement can be found here )Given n positive integers a_1, a_2, ..., a_n , find the sum of the GCD (greatest common divisor) of every possible pair of these integers.

Input:

The first line contains the number of test cases t\:(1 < t < 100). Each test case consists of one line that contains the number n\:(1 < n < 100), followed by n positive integers. The input integers do not exceed 10^6.

Output:

Given n positive integers a_1, a_2, ..., a_n, find the sum of the GCD (greatest common divisor) of every possible pair of these integers.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source 2013 ACM-ICPC Asia Phuket Regional Programming Contest, Practice Session, November 21

Solution:

def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a % b)

t = int(input())
for _ in range(t):
    a = list(map(int, input().split()))
    m = a[0]
    a = a[1:]
    ans = 0
    for i in range(m):
        for j in range(i + 1, m):
            ans += gcd(a[i], a[j])
    print(ans)
