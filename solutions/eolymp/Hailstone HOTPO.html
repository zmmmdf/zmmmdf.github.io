Hailstone HOTPO

Statement:

( The original statement can be found here )The hailstone sequence is formed in the following way: It is conjectured that for any positive integer number n , the sequence will always end in the repeating cycle: 4, 2, 1, 4, 2, 1, ... . Suffice to say, when n = 1 , we will say the sequence has ended. Write a program to determine the largest value in the sequence for a given **n**.

Input:

The first line contains the number of data sets t~(1 \le t \le 10^5) that follow. Each data set should be processed identically and independently.Each data set consists of a single line of input consisting of two space separated decimal integers. The first integer is the data set number. The second integer is n~(1 \le n \le 10^5), which is the starting value.

Output:

The hailstone sequence is formed in the following way:

Constraints:

Time limit 3 seconds
Memory limit 128 MiB
Source 2012 Greater New York Region Programming Contest, October 28, Problem A

Solution:

#include <iostream>
using namespace std;

int main() {
    int t;
    cin >> t;
    
    while (t--) {
        int c, n;
        cin >> c >> n;
        int mx = n;
        
        while (n != 1) {
            if (n % 2 == 0) {
                n /= 2;
            } else {
                n = n * 3 + 1;
            }
            mx = max(mx, n);
        }
        
        cout << c << " " << mx << endl;
    }
    
    return 0;
}
