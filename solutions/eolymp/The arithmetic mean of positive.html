The arithmetic mean of positive

Statement:

( The original statement can be found here )The sequence of real numbers is given. Find the arithmetic mean of positive numbers.

Input:

First line contains amount n (0 < n ≤ 100) of real numbers. Next line contains n real numbers, each of them is not greater than 100 by absolute value.

Output:

The sequence of real numbers is given. Find the arithmetic mean of positive numbers.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source SFE-2010 Variant 10
Show problem classificationbeginners

Solution:

n = int(input())
ns = list(map(float, input().split()))
res = 0
c = 0
for i in range(n):
    if ns[i] > 0:
        res += ns[i]
        c += 1
if c == 0: print("Not Found")
else: print(f"{res/c:.2f}")
