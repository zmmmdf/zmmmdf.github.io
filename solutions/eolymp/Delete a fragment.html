Delete a fragment

Statement:

( The original statement can be found here )Given a string of Latin letters and spaces. Delete in it all characters with indices from n to m inclusive. Indexation starts from 0 .

Input:

The first line contains text of no more than 1000 Latin letters and spaces. Second line contains two integers n and m (n ≤ m).

Output:

Given a string of Latin letters and spaces. Delete in it all characters with indices from n to m inclusive. Indexation starts from 0.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Author Matviychuk Sergiy Volodymyrovych
Source "ABC programming"

Solution:

a = input()
m, n = map(int, input().split())
print(a[:m] + a[n+1:])
