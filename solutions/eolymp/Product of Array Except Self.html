Product of Array Except Self

Statement:

( The original statement can be found here )Given an array in of n integers. Build an array out such that out_i is equal to the product of all the elements of in except in_i .

Input:

The first line contains number n~(1 < n \le 10^6). The second line contains n integers, each number is not greater than 100 by absolute value.

Output:

Given an array in of n integers. Build an array out such that out_i is equal to the product of all the elements of in except in_i.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Show problem classificationbeginners, simple-mathematics

Solution:

#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> a(n);
    long long product = 1;
    int zeroCount = 0;
    for (int i = 0; i < n; i++) {
        cin >> a[i];
        if (a[i] == 0) {
            zeroCount++;
            continue;
        }
        product *= a[i];
    }
    vector<int> out(n);
    if (zeroCount > 1) {
        // Bütün out elementləri 0 olacaq.
    } else if (zeroCount == 1) {
        for (int i = 0; i < n; i++) {
            if (a[i] == 0) {
                out[i] = product;
                break;
            }
        }
    } else {
        for (int i = 0; i < n; i++) {
            out[i] = product / a[i];
        }
    }
    for (int i = 0; i < n; i++) {
        cout << out[i] << " ";
    }
    cout << endl;
    return 0;
}
