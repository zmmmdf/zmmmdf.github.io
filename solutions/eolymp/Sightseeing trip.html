Sightseeing trip

Statement:

( The original statement can be found here )The pupils from 10-B class decided in the autumn holiday to go on an excursion to the capital. Knowing the number of boys n and girls m , determine how many rooms you need to book in a hotel, if each room has k beds and it is forbidden to settle boys and girls together.

Input:

One line contains three numbers n, m, k~(n, m, k \le 100).

Output:

The pupils from 10-B class decided in the autumn holiday to go on an excursion to the capital. Knowing the number of boys n and girls m, determine how many rooms you need to book in a hotel, if each room has k beds and it is forbidden to settle boys and girls together.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Show problem classificationif-then

Solution:

#include <iostream>

using namespace std;

int main()
{
	int n, m, k;
	cin >> n >> m >> k;
	cout << (n + k - 1) / k + (m + k - 1) / k;
	return 0;
}
