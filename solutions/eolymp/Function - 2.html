Function - 2

Statement:

( The original statement can be found here )Implement a function f(x) = \sqrt{x} + 2 \cdot x + sin(x) .

Input:

Each line contains one real number x.

Output:

Implement a function f(x) = \sqrt{x} + 2 \cdot x + sin(x).

Constraints:

Time limit 1 second
Memory limit 128 MiB
Author Mykhailo Medvediev
Source Programming language C

Solution:

import math
def fx(x):
    return (x**.5) + 2*x + math.sin(x)
ans = []
a = " "
try:
    while a:
        a = float(input())
        ans.append(fx(a))
except:
    for i in ans:
        print(f"{i:.4f}")
