Tricky Sorting

Statement:

( The original statement can be found here )The sequence of numbers is given. Arrange them in non-decreasing order of the last digit, and in the case of equality of last digits — arrange the numbers in non-decreasing order.

Input:

The first line contains number n~(1 \le n \le 100), and next lines contain the positive integers not greater than 32000.

Output:

The sequence of numbers is given. Arrange them in non-decreasing order of the last digit, and in the case of equality of last digits — arrange the numbers in non-decreasing order.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Show problem classificationsort-and-sequence

Solution:

n = int(input())
nums = []
for i in range(n):
    nums.append(int(input()))

nums.sort(key=lambda x: (x % 10, x))

for num in nums:
    print(num)
