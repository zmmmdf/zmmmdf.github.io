Binary search

Statement:

( The original statement can be found here )Sorted array of n integers is given. You must answer q queries: whether the given number x is in the array.

Input:

First line contains two numbers n and q\:(n, q \le 10^6). Second line contains n integers sorted in increasing order. Each of the next q lines contains value of x. The numbers in array do not exceed 10^9 by absolute value.

Output:

Sorted array of n integers is given. You must answer q queries: whether the given number x is in the array.

Constraints:

Time limit 5 seconds
Memory limit 256 MiB
Show problem classificationbinary-search

Solution:

#include <bits/stdc++.h>

using namespace std;

int main() {
    int n, q;
    cin >> n >> q;

    int arr[n];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    sort(arr, arr + n); 

    for (int i = 0; i < q; i++) {
        int x;
        cin >> x;

        if (binary_search(arr, arr + n, x)) {
            cout << "YES" << endl;
        } else {
            cout << "NO" << endl;
        }
    }

    return 0;
}
