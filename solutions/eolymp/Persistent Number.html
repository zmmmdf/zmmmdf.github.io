Persistent Number

Statement:

( The original statement can be found here )Given a number x , we define a function p(x) as the product of the digits of x . We can then form a sequence x, p(x), p(p(x))... . The persistence of x is then defined as the index ( 0 -based) of the first single digit number in the sequence. For example, using 99 , we get the sequence 99, 9 \cdot 9 = 81, 8 \cdot 1 = 8 . Thus, the persistence of 99 is 2 . You will be given n , and you must find its persistence.

Input:

Each line contains one integer n\:(0 \le n \le 2 \cdot 10^9).

Output:

Given a number x, we define a function p(x) as the product of the digits of x. We can then form a sequence x, p(x), p(p(x))... . The persistence of x is then defined as the index (0-based) of the first single digit number in the sequence. For example, using 99, we get the sequence 99, 9 \cdot 9 = 81, 8 \cdot 1 = 8. Thus, the persistence of 99 is 2. You will be given n, and you must find its persistence.

Constraints:

Time limit 1 second
Memory limit 128 MiB

Solution:

while True:
    try:
        a = int(input())
    except:
        break
    a = str(a)
    if a:
        res = 0
        while len(a) != 1:
            s = 1
            for i in range(len(a)):
                s *= int(a[i])
            a = str(s)
            res += 1
        print(res)
    else: break
