Bitwise Equations

Statement:

( The original statement can be found here )You are given two positive integers x and k . Find the k -th smallest positive integer y (where k is 1 -based) for which the following equation holds: x + y = x | y where '|' denotes the bitwise OR operator.

Input:

Each line is a separate test case which contains two integers x and k (1 ≤ x, k ≤ 2*10^9).

Output:

You are given two positive integers x and k. Find the k-th smallest positive integer y (where k is 1-based) for which the following equation holds:

Constraints:

Time limit 1 second
Memory limit 64 MiB

Solution:

#include <bits/stdc++.h>
using namespace std;

long long kthSmallest(long long x, long long k)
{

    long long y = 0;

    for(int i = 0; k; i++)
    {
        if (x >> i & 1) continue;
        if (k % 2) y = y | (1LL << i);
        k >>= 1;
    }

    return y;

}
int main()
{
    long long x, k;
    while(cin>>x>>k)
    {
        cout<<kthSmallest(x, k)<<endl;
    }
}
