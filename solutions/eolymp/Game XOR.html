Game XOR

Statement:

( The original statement can be found here )Consider the game for two players. Given a sequence of 0 and 1 . At each turn a player must choose any two adjacent elements and to replace them in the sequence with one number - the result of the bitwise addition modulo 2 ( XOR ). The players take turns until there is one element left. If it will be 0 then wons the one who makes the move first, otherwise the second player wins. Determine the winner, provided that both play optimally.

Input:

The first line contains an integer n (1 ≤ n ≤ 1000) - number of elements in the sequence. The second line contains the elements themselves - 0 and 1 separated by spaces.

Output:

Consider the game for two players. Given a sequence of 0 and 1. At each turn a player must choose any two adjacent elements and to replace them in the sequence with one number - the result of the bitwise addition modulo 2 (XOR). The players take turns until there is one element left. If it will be 0 then wons the one who makes the move first, otherwise the second player wins. Determine the winner, provided that both play optimally.

Constraints:

Time limit 1 second
Memory limit 128 MiB
Author A. Milanin
Source ACM, Ukraine, First Stage, 09.04.2011

Solution:

#include <iostream>
using namespace std;


int main()
{
    int q, ans, e;
    cin >> q >> ans;
    while(q != 1){
        cin >> e;
        ans ^= e;
        q--;
    }
    if (ans % 2) cout << "Second" << endl;
    else cout << "First" << endl;
}
