Strange Sequence

Statement:

( The original statement can be found here )From the given number n , the following sequence S is obtained: = n , = f ( ), i ≥ 1 Here f ( x ) is equal to the difference between the numbers obtained by rearranging the digits of the number x in descending and ascending order. For example: f ( 2214 ) = 4221 − 1224 = 2997 Note that leading zeros are not counted. Find the value of .

Input:

Two integers n (0 ≤ n ≤ 10^9) and k (0 ≤ k ≤ 10^5).

Output:

From the given number n, the following sequence S is obtained:

Constraints:

Time limit 1 second
Memory limit 128 MiB
Source 2022 Azerbaijan Republic Olympiad, Semifinals, March 8

Solution:

def f(x):
    digits = sorted(str(x))
    increasing = int("".join(digits))
    decreasing = int("".join(digits[::-1]))
    return decreasing - increasing

n, k = map(int, input().split())
s = n
for i in range(k):
    s = f(s)
print(s)
